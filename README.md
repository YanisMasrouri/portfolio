# Portfolio Yanis Masrouri

Portfolio personnel responsive développé avec **React**, **TypeScript**, **Vite** et **Tailwind CSS**.

Le site présente une double identité professionnelle :

- **Ingénieur** : développement logiciel, architecture, interfaces industrielles, OPC UA, WPF, MVVM.
- **Créatif** : montage vidéo, narration, direction visuelle, identité graphique et projets visuels.

L’objectif du projet est d’avoir un portfolio moderne, maintenable, bilingue et facilement évolutif.

## Aperçu

Le portfolio est conçu comme une page immersive avec :

- une section d’accueil ;
- une navigation responsive ;
- une séparation claire entre les profils **Ingénieur** et **Créatif** ;
- une grille de projets automatique ;
- un système bilingue français / anglais ;
- une section contact ;
- une architecture simple à maintenir.

Le design repose sur une ambiance sombre, technique et cinématique.

---

## Objectifs du projet

Ce portfolio a pour objectif de présenter :

- mon profil d’ingénieur logiciel ;
- mes projets techniques ;
- mes compétences en développement logiciel ;
- mes projets créatifs ;
- mes capacités en interface, narration visuelle et direction artistique ;
- une identité professionnelle hybride : rigueur technique et sens créatif.

Le site doit rester :

- responsive ;
- rapide ;
- facile à modifier ;
- facile à maintenir ;
- clair sur ordinateur comme sur téléphone.
